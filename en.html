    Install the extension in Firefox

  • 1. Download the latest archive with the extension from our GitHub repository.
  • 2. Unzip the Extension into a new folder.
  • 3. Open a "New Tab" and go to this address: about:debugging#/runtime/this-firefox.
  • 4. Next, click the Download temporary add-on button.
  • 5. Go to the folder where you unpacked the extension and select the manifest.json.
  • Ready! You can use the extension.
  • *You will need to do this every time you start the browser.

    Install the extension in Google Chrome

  • 1. Download the latest archive with the extension from our GitHub repository.
  • 2. Unzip the Extension into a new folder.
  • 3. Open a "New Tab" and go to this address: chrome://extensions.
  • 4. In the upper right corner, turn on "Developer Mode".
  • 5. Next, click on the button Download the unpacked file.
  • 6. Select the folder where you unzipped the extension.
  • Ready! You can use the extension.
